# 发法牛 - 发布的想法都很牛！

📝 一款简洁高效的本地优先笔记应用，让记录想法变得轻松自如。

## ✨ 核心功能

### 📝 笔记记录
- 快速记录想法，支持富文本内容
- 图片上传（每条笔记最多 9 张图片）
- 自动保存草稿，不用担心内容丢失
- 支持 `Ctrl+Enter` 快捷发送

### 🏷️ 标签系统
- 使用 `#标签名` 格式自动识别标签
- 标签智能高亮，点击可快速筛选
- 标签支持三种排序方式：按数量 / 按时间 / 按名称
- 支持升序/降序切换

### ✅ 状态管理
- 已处理/未处理状态切换
- 一键标记全部为已处理
- 处理时间自动记录
- 状态筛选（全部 / 未处理 / 已处理）

### 📊 数据管理
- 本地 IndexedDB 存储，离线可用
- 导入/导出数据备份（JSON 格式）
- 存储空间实时查看

### 🔍 搜索与筛选
- 全文搜索
- 按标签筛选
- 按状态筛选
- 清除筛选快捷方便

## 🚀 快速开始

```bash
# 克隆项目
git clone https://github.com/Niorghini/ffn-web.git

# 安装依赖
cd ffn-web
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build
```

打开 http://localhost:5174 即可使用。

## 🛠️ 技术栈

- **前端框架**: React 18
- **构建工具**: Vite 5
- **样式**: Tailwind CSS
- **数据库**: Dexie.js (IndexedDB 封装)
- **图标**: Lucide React
